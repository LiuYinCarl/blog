## 查看动态库导出函数列表

**1、objdump**

``` bash
objdump -tT xxx.so
```

**2、nm**

``` bash
nm -D xxx.so
```

​-D 或 -dynamic 选项表示：显示动态符号。该选项仅对于动态库有意义。

## Linux 多个静态库打包成一个静态库

[Linux 多个静态库打包成一个静态库](https://blog.csdn.net/listener51/article/details/104400299)

``` bash
# 将libabc.a libxyz.a 打包成libaz.a
ar -rcT libaz.a libabc.a libxyz.a
```
